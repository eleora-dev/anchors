Privacy Policy

Eleòra Anchors is a Chromium-based browser extension that provides quick access to bookmarks, clean page reload, cache clearing, sanitized URL copy, and incognito/private navigation.

This extension does not collect, store, transmit, sell, or share any personal data. All operations are performed locally within your browser. No user data ever leaves your device.

The extension does not store or persist any user data, including:

  • Bookmarks data
  • Visited URLs
  • Clipboard contents
  • Browsing history
  • Cache or local site data

Data handling:

  • Bookmarks are accessed only to display them in the popup.
  • The current tab URL is accessed only when the user copies a sanitized URL or performs a clean reload.
  • Browsing data is accessed only when the user explicitly clicks a cleaning action.
  • Clipboard access is used only to copy the sanitized URL requested by the user.
  • All operations are performed in memory and discarded immediately after use.

The following permissions are required:

  • bookmarksRead and display bookmarks in the popup.
  • tabsRead the current tab URL when needed.
  • windowsOpen and manage browser windows, including incognito/private windows.
  • browsingDataClear cache and selected browsing data when requested by the user.
  • clipboardWriteCopy sanitized URLs to the clipboard.
  • faviconShow site icons next to bookmarks.

No analytics. No tracking. No external services or network requests.

Italiano

Informativa sulla Privacy

Eleòra Anchors è un'estensione per browser basati su Chromium che fornisce accesso rapido ai preferiti, ricarica pulita della pagina, pulizia della cache, copia dell'URL senza parametri di tracking e navigazione in incognito/privata.

Questa estensione non raccoglie, salva, trasmette, vende o condivide alcun dato personale. Tutte le operazioni vengono eseguite localmente nel browser. Nessun dato dell'utente lascia il dispositivo.

L'estensione non salva né conserva alcun dato dell'utente, inclusi:

  • Dati dei preferiti
  • URL visitati
  • Contenuto degli appunti
  • Cronologia di navigazione
  • Cache o dati locali dei siti

Gestione dei dati:

  • I preferiti vengono letti solo per mostrarli nel popup.
  • L'URL della scheda corrente viene letto solo quando l'utente copia un URL pulito o esegue una ricarica pulita.
  • I dati di navigazione vengono accessi solo quando l'utente clicca esplicitamente un'azione di pulizia.
  • L'accesso agli appunti viene usato solo per copiare l'URL pulito richiesto dall'utente.
  • Tutte le operazioni vengono eseguite in memoria e scartate immediatamente dopo l'uso.

I seguenti permessi sono necessari:

  • bookmarksLeggere e mostrare i preferiti nel popup.
  • tabsLeggere l'URL della scheda corrente quando necessario.
  • windowsAprire e gestire finestre del browser, incluse finestre in incognito/private.
  • browsingDataPulire cache e dati di navigazione selezionati quando richiesto dall'utente.
  • clipboardWriteCopiare URL puliti negli appunti.
  • faviconMostrare le icone dei siti accanto ai preferiti.

Nessuna analisi. Nessun tracciamento. Nessun servizio esterno né comunicazione di rete.
